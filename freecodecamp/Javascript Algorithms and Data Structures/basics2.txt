# object in JavaScript

var cat = {
	"name": "Whiskers",
	"legs": 4,
	"tails": 2,
	"pets": ["cat", "dog"],
	5: "five",
	something: "nothing"
};

* we can use . notation to access values from an object
eg. alert(cat.name);

* we can use [] notation to access values from an object
eg. alert(cat[name]);
If the property has spaces, using quotes while accessing the property are mandatory
